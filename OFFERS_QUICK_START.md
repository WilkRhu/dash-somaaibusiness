# Guia Rápido - Módulo de Ofertas

## 🚀 Início Rápido

### 1. Acessar o Módulo

```
Dashboard → Sidebar → Ofertas
```

### 2. Criar sua Primeira Oferta

1. Clique em **"+ Nova Oferta"**
2. Selecione um produto do inventário
3. Defina o título (ex: "Promoção de Verão")
4. Defina o preço promocional (menor que o original)
5. Escolha o período de validade
6. Clique em **"Criar Oferta"**

### 3. Gerenciar Ofertas

#### Ativar/Desativar
- Clique no botão ▶️ para ativar
- Clique no botão ⏸️ para pausar

#### Editar
- Clique em "Editar" na linha da oferta
- Modifique os campos desejados
- Salve as alterações

#### Ver Analytics
- Clique no ícone 📊
- Visualize métricas de performance
- Analise insights automáticos

## 📊 Entendendo as Métricas

### Visualizações
Quantas vezes a oferta foi vista no app Consumer

### Salvamentos
Quantos usuários salvaram a oferta para ver depois

### Conversões
Quantas vendas foram realizadas através da oferta

### Taxa de Conversão
```
(Conversões / Visualizações) × 100
```

### Taxa de Salvamento
```
(Salvamentos / Visualizações) × 100
```

## 🎯 Dicas para Ofertas de Sucesso

### 1. Título Atrativo
✅ "Promoção Relâmpago - 50% OFF"
❌ "Desconto"

### 2. Descrição Clara
✅ "Válido apenas hoje! Aproveite enquanto durar o estoque"
❌ "Oferta especial"

### 3. Desconto Significativo
- Mínimo recomendado: 15%
- Ideal: 20-40%
- Promoções relâmpago: 50%+

### 4. Período Estratégico
- Promoções curtas (1-3 dias): Senso de urgência
- Promoções longas (1 semana+): Maior alcance
- Fins de semana: Maior tráfego

### 5. Produtos Populares
- Escolha produtos com boa rotatividade
- Evite produtos próximos ao vencimento
- Considere produtos sazonais

## 📈 Interpretando Analytics

### Alta Performance
```
Taxa de Conversão > 10%
Taxa de Salvamento > 20%
```
✅ Oferta está funcionando muito bem!

### Performance Média
```
Taxa de Conversão: 5-10%
Taxa de Salvamento: 10-20%
```
⚠️ Considere ajustar preço ou descrição

### Baixa Performance
```
Taxa de Conversão < 5%
Muitas visualizações mas poucas vendas
```
❌ Revise a oferta:
- Preço ainda alto?
- Descrição clara?
- Produto desejado?

## 🔧 Solução de Problemas

### Não consigo ativar a oferta
- Verifique se está dentro do período de validade
- Confirme que o produto ainda existe no inventário
- Verifique se não atingiu o limite do seu plano

### Oferta não aparece no app Consumer
- Confirme que está ativa
- Verifique se o estabelecimento está ativo
- Aguarde alguns minutos para sincronização

### Erro ao criar oferta
- Preço da oferta deve ser menor que o original
- Data de início deve ser anterior à data de término
- Todos os campos obrigatórios devem estar preenchidos

## 📱 Integração com App Consumer

### Como Funciona

1. **Usuário abre o app** → Vê ofertas próximas
2. **Visualiza a oferta** → Contador de views aumenta
3. **Salva a oferta** → Contador de salvamentos aumenta
4. **Realiza compra** → Contador de conversões aumenta

### Visibilidade

Ofertas são mostradas para usuários:
- Dentro do raio de alcance do estabelecimento
- Com a oferta ativa
- Durante o período de validade

## 🎨 Boas Práticas de UI

### Use Imagens Atrativas
- Produtos com imagens vendem 3x mais
- Use fotos de alta qualidade
- Mostre o produto em uso

### Destaque o Desconto
- Percentual de desconto é calculado automaticamente
- Aparece em destaque no card da oferta
- Chama atenção do consumidor

### Crie Senso de Urgência
- "Últimas unidades!"
- "Válido apenas hoje!"
- "Termina em X dias"
