## 📱 Em Dispositivos Móveis

### Se o Menu Estiver Fechado

```
1. Clique no ícone ☰ (menu)
   ↓
2. O menu abrirá
   ↓
3. Procure por "💳 Mercado Pago"
   ↓
4. Clique nele
```

### Se o Menu Estiver Aberto

```
1. Procure por "💳 Mercado Pago"
   ↓
2. Clique nele
```

Se você quiser ir direto, digite na barra de endereço:
