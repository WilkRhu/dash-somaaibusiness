# Como Acessar Mercado Pago

## 📍 Localização

O botão de Mercado Pago está localizado na página de detalhes de cada estabelecimento.

---

## 🚀 Passo a Passo

### Passo 1: Acesse Estabelecimentos
1. No menu lateral, clique em **Estabelecimentos**
2. Você verá uma lista de todos os seus estabelecimentos

### Passo 2: Selecione um Estabelecimento
1. Clique no estabelecimento que deseja configurar
2. A página de detalhes será aberta

### Passo 3: Clique no Botão Mercado Pago
Na página de detalhes, você verá 4 botões:
- **Fidelidade** - Configure o programa de fidelidade
- **Editar** - Edite os dados do estabelecimento
- **Mercado Pago** - Configure a integração (novo botão)
- **Desativar** - Desative o estabelecimento

Clique no botão **Mercado Pago**

### Passo 4: Pronto!
Você será redirecionado para a página de configuração do Mercado Pago

```

    ↓
Estabelecimentos
    ↓

    ↓

    ↓
Página de Configuração do Mercado Pago
```
