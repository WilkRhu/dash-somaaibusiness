# 🗺️ Como Acessar a Integração Mercado Pago

## 📍 Localização

A página de integração do Mercado Pago está localizada em:

**URL:** `/establishments/mercadopago`

---

## 🧭 Passo a Passo para Chegar Lá

### Opção 1: Pelo Menu Lateral (Recomendado)

1. **Abra o sistema** - Acesse o dashboard do SomaAI Business
2. **Procure no menu lateral esquerdo** por:
   - 🏪 **Estabelecimentos**
   - 💳 **Mercado Pago** (novo item)
3. **Clique em "💳 Mercado Pago"**
4. **Pronto!** Você está na página de configuração

### Opção 2: Digitando a URL Diretamente

1. **Na barra de endereço do navegador**, digite:
   ```
   /establishments/mercadopago
   ```
2. **Pressione Enter**
3. **Pronto!** Você será redirecionado para a página

### Opção 3: Pelo Menu de Estabelecimentos

1. **Clique em "Estabelecimentos"** no menu lateral
2. **Procure por um link ou botão** que diga "Integração Mercado Pago"
3. **Clique nele**
4. **Pronto!** Você está na página de configuração

## 📱 Em Dispositivos Móveis

### Se o Menu Estiver Fechado

1. **Clique no ícone de menu** (☰) no canto superior esquerdo
2. **O menu lateral abrirá**
3. **Procure por "💳 Mercado Pago"**
4. **Clique nele**

### Se o Menu Estiver Aberto

1. **Procure por "💳 Mercado Pago"** no menu
2. **Clique nele**
3. **Pronto!**

---

## 🔍 Dicas de Navegação

### Procurando o Item?

Se você não conseguir encontrar "💳 Mercado Pago" no menu:

1. **Verifique se você está logado** - Você precisa estar autenticado
2. **Verifique suas permissões** - Você precisa ter permissão para gerenciar estabelecimentos
3. **Atualize a página** - Pressione F5 ou Ctrl+R
4. **Limpe o cache** - Limpe o cache do navegador e tente novamente

### Atalho de Teclado

Se você souber a URL, pode usar:

1. **Pressione Ctrl+L** (ou Cmd+L no Mac) para focar na barra de endereço
2. **Digite:** `/establishments/mercadopago`
3. **Pressione Enter**

## 🆘 Problemas de Acesso

### Não Vejo o Item no Menu

**Possíveis causas:**
- Você não tem permissão
- O item não foi carregado
- Você está em outro estabelecimento

**Solução:**
1. Verifique suas permissões
2. Atualize a página (F5)
3. Limpe o cache do navegador
4. Tente novamente

### Erro ao Acessar a Página

**Possíveis causas:**
- Você não está autenticado
- A página não existe
- Erro de servidor

**Solução:**
1. Verifique se você está logado
2. Tente acessar a URL diretamente
3. Atualize a página
4. Contate o suporte

### Página em Branco

**Possíveis causas:**
- Página ainda carregando
- Erro de JavaScript
- Problema de conexão

**Solução:**
1. Aguarde alguns segundos
2. Atualize a página (F5)
3. Limpe o cache
4. Tente em outro navegador
