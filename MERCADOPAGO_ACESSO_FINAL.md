# 💳 Como Acessar Mercado Pago - Versão Final

## 🎯 Resposta Rápida

### Caminho:
```
Menu Lateral → Estabelecimentos → Selecione um → Clique em "Mercado Pago"
```

---

## 📍 Localização do Botão

O botão **Mercado Pago** está na página de detalhes de cada estabelecimento, junto com os botões:
- Fidelidade
- Editar
- Desativar

---

## 🚀 Passo a Passo

### 1. Abra o Menu Lateral
Procure pelo menu no lado esquerdo da tela

### 2. Clique em "Estabelecimentos"
```
Estabelecimentos
```

### 3. Selecione um Estabelecimento
Clique no estabelecimento que deseja configurar

### 4. Clique no Botão "Mercado Pago"
Na página de detalhes, você verá os botões:
```
[Fidelidade] [Editar] [Mercado Pago] [Desativar]
                           ↑
                      CLIQUE AQUI
```

### 5. Pronto!
Você será redirecionado para a página de configuração do Mercado Pago
