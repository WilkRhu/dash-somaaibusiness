# 💳 Como Acessar a Integração Mercado Pago

## 🎯 Resposta Rápida

### Caminho Mais Rápido:
```
Menu Lateral → 🏪 Estabelecimentos → 💳 Mercado Pago
```

### URL Direta:
```
/establishments/mercadopago
```

---

## 📍 Passo a Passo

### Passo 1: Abra o Sistema
Acesse o dashboard do SomaAI Business

### Passo 2: Procure o Menu Lateral
No lado esquerdo da tela, você verá o menu com vários itens

### Passo 3: Clique em "Estabelecimentos"
```
🏪 Estabelecimentos
```

### Passo 4: Clique em "Mercado Pago"
```
💳 Mercado Pago
```

### Passo 5: Pronto!
Você está em `/establishments/mercadopago`

## 🔗 Atalhos

### Atalho 1: Digitando a URL
1. Clique na barra de endereço
2. Digite: `/establishments/mercadopago`
3. Pressione Enter

### Atalho 2: Atalho de Teclado
1. Pressione `Ctrl+L` (ou `Cmd+L` no Mac)
2. Digite: `/establishments/mercadopago`
3. Pressione Enter

---

## 📱 Em Dispositivos Móveis

### Se o Menu Estiver Fechado
1. Clique no ícone ☰ (menu)
2. Procure por "💳 Mercado Pago"
3. Clique nele

### Se o Menu Estiver Aberto
1. Procure por "💳 Mercado Pago"
2. Clique nele

## 🆘 Problemas?

### Não Vejo o Item no Menu
- Atualize a página (F5)
- Limpe o cache do navegador
- Verifique suas permissões

### Erro ao Acessar
- Verifique se você está logado
- Tente acessar a URL diretamente
- Atualize a página

### Página em Branco
- Aguarde alguns segundos
- Atualize a página (F5)
- Limpe o cache
